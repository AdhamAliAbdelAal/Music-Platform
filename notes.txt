--------------commands----------------
pip list : list all packages installed in the current environment
Get-Command === which
python manage.py runserver: run server
python manage.py createsuperuser: create sudo
python manage.py startapp app_name: create app
python manage.py migrate: commit&push migrations(run files)
python manage.py makemigrations : add migrations to something like staging area(make files)
python manage.py sqlmigrate app_name migration_num: show sql command executed
python manage.py shell: run database shell
django-env\Scripts\activate activate virtual environment
pip install django-crispy-forms crispy installation
